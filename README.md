# Mini Search Engine (Граф страниц)

## 📌 Описание проекта
Данный проект представляет собой мини-поисковик, реализованный на языке Java.  
Он моделирует структуру сайта в виде графа и выполняет поиск страниц с использованием алгоритма обхода.

Проект демонстрирует применение теории графов в реальных задачах поиска информации.

---

## 🎯 Цель работы
Изучить применение графов для организации поиска данных и реализовать простой поисковый механизм.

---

## 🧠 Основная идея
Сайт представлен как граф:
- страницы — вершины
- ссылки — рёбра

Поиск выполняется путем обхода этого графа.

---

## ⚙️ Используемые технологии
- Java
- Java Swing (GUI)
- Структуры данных (Map, List, Queue, Set)

---

## 🔍 Алгоритм поиска
В проекте реализован алгоритм **BFS (Breadth-First Search)** — поиск в ширину.

### Принцип работы:
1. Начинаем с главной страницы
2. Переходим ко всем соседним страницам
3. Используем очередь для хранения вершин
4. Избегаем повторного посещения

---

## 🖥️ Функционал
- Ввод поискового запроса
- Обход графа страниц
- Вывод результата поиска
- Отображение пути обхода
- Переключение тёмной/светлой темы

---

## 📂 Структура графа (пример)
Главная → Новости, Контакты, О нас
Новости → Спорт, Технологии
Контакты → Email, Телефон

---

## ▶️ Запуск программы
1. Открыть проект в IDE (IntelliJ IDEA / Eclipse)
2. Запустить класс `SearchEngine`
3. Ввести название страницы в поле поиска
4. Нажать кнопку "Поиск"

---

## 📊 Результат
Программа выводит:
- порядок обхода страниц
- результат поиска (найдено / не найдено)

---

## 🚀 Возможности для улучшения
